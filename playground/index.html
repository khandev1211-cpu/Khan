Khan, in your browser

A language built from nothing but a C compiler: its own lexer, parser, bytecode compiler, and stack-based VM. No server runs this code — the same engine that compiles to a native binary was compiled to WebAssembly and is running the lexer/parser/compiler/VM pipeline locally, in this tab.
